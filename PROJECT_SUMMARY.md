## 🚀 Quick Start

1. **Make setup script executable:**
   ```bash
   chmod +x scripts/setup.sh
   ```

2. **Run setup script:**
   ```bash
   ./scripts/setup.sh
   ```

3. **Start development:**
   ```bash
   # Option 1: Docker (recommended)
   docker-compose up -d
   
   # Option 2: Manual
   # Backend
   cd backend && source venv/bin/activate && uvicorn main:app --reload
   
   # Frontend  
   cd frontend && npm start
   ```
